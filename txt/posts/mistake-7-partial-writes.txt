  SPECIAL -- IPC MISTAKE: PARTIAL WRITES
  Mistake 7: Partial writes
  ============================================================================================


     `write()` does not guarantee all bytes are written in one go. It returns the number of bytes written. Robust code must loop until `bytes_written == total_len`.
