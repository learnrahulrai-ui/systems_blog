  SPECIAL -- IPC MISTAKE: THE TIME_WAIT / EADDRINUSE BIND BUG
  Mistake 11: The TIME_WAIT / EADDRINUSE bind bug
  ============================================================================================


      If a server closes a connection and exits, the OS holds the port in a TIME_WAIT state to catch delayed packets. Restarting the server instantly throws "Address already in use". The fix is `setsockopt(..., SO_REUSEADDR, ...)`. Note: Cygwin/Windows hides this by allowing rebinds, but Linux strictly crashes without it.
